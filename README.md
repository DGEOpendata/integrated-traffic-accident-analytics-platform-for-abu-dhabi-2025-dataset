# Integrated Traffic Accident Analytics Platform - Abu Dhabi 2025 Dataset

## Overview
The Integrated Traffic Accident Analytics Platform is designed to provide actionable insights from the Abu Dhabi Traffic Accident Dataset for 2025. It features modules for hotspot detection, correlation analysis, and injury statistics, assisting policymakers, urban planners, researchers, and other stakeholders.

## Features
- **Heatmap Generator**: Visualize accident hotspots across cities and streets.
- **Correlation Analyzer**: Identify relationships between weather conditions and accident rates.
- **Injury Statistics Module**: Generate descriptive statistics on accident-related injuries.

## Installation
1. Clone the repository:
   bash
   git clone https://github.com/your-repository/traffic-accident-analytics.git
   
2. Navigate to the project directory:
   bash
   cd traffic-accident-analytics
   
3. Install required dependencies:
   bash
   pip install -r requirements.txt
   

## Usage
1. Place the `Traffic_Accident_Abu_Dhabi_2025.xlsx` dataset file in the project directory.
2. Run the Python script:
   bash
   python analyze_accidents.py
   
3. Follow the prompts to generate visualizations and insights.
